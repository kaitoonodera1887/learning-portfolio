### コマンドメモ

- 有効化: `source .venv/bin/activate` (mac/Linux) / `.\.venv\Scripts\Activate.ps1` (PowerShell)
- pip 更新: `python -m pip install --upgrade pip setuptools`
- 実行: `python hello.py`
